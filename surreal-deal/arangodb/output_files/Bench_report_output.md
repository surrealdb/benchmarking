
# Surreal bench

This benchmark compares SurrealDB and ArangoDB performance across a variety of CRUD queries.

It consists of the following tables:

| Table name     |   Number of records |
|----------------|---------------------|
| person_amount  |                1000 |
| product_amount |                1000 |
| order_amount   |               10000 |
| artist_amount  |                 500 |
| review_amount  |                2000 |
